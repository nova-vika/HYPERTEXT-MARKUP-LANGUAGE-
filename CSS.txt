------------------------------------------------------- CSS display ---------------------------------------------------------


DISPLAY PROPERTY :

>>>> The CSS display property defines how an HTML element is displayed on the web page.
>>>> It controls the layout behavior of elements such as whether they appear on a new line, share space, or are hidden.

--->> Syntax:
         display: value;

>>>> Common display values used in web development:

                *  block
                * inline  
                * inline-block
                * none

<><><> 1. DISPLAY: BLOCK :

--->> Definition:
           >. Block-level elements start on a new line and take the full width of their parent container.

>>>> Characteristics:

**** Always starts on a new line

**** Takes 100% available width by default

**** Width and height can be applied

**** Margin and padding work on all sides

**** Examples of block elements:

           div, p, h1â€“h6, section, header, footer


<><><> 2. DISPLAY: INLINE :

---->>> Definition:
            >. Inline elements do not start on a new line and take only as much width as their content.

>>>> Characteristics:

**** do not start on a new line

**** Width and height cannot be set

**** Margin works only left and right

**** Padding works but may overlap vertically

**** Examples of inline elements:
             span, a, strong, em


<><><> 3. DISPLAY: INLINE-BLOCK :

>>>> Definition:
        >. Inline-block elements behave like inline elements but allow width and height.

>>>> Characteristics:

**** Stays in the same line

**** Width and height can be set

**** Margin and padding work on all sides

**** Example:
.button {
display: inline-block;
width: 150px;
height: 40px;
}


<><><> 4. DISPLAY: NONE :

>>> Definition:
      >. The element is completely removed from the layout and not visible.

>>> Characteristics:

**** Element is hidden

**** Takes no space

**** Not accessible visually

**** Example:
.hidden {
display: none;
}



-------------------------------------------------- CSS positioning ---------------------------------------------------------


>>>>>>>>>  The CSS position property defines how an element is positioned in the document and how properties like
 top, right, bottom, and left affect it.

>.>>>> Syntax:

position: value;

>>>> Position values:

          ** static, relative, absolute, fixed, sticky

1. POSITION: STATIC (default):

>> Definition:
      >. Static is the default position for all HTML elements.

>>> Characteristics:

**** Element follows normal document flow

**** top, right, bottom, left do NOT work

**** No special positioning behavior

**** Example:
div {
position: static;
}

>>>> Note:
       >> All elements are static by default unless changed.

2. POSITION: RELATIVE :

>>> Definition:
       >. Relative positioning moves the element relative to its original position.

>>>> Characteristics:

**** Element stays in normal flow

**** Space is still reserved

**** top, left, right, bottom work

**** Used as a reference for absolute elements

**** Example:
.box {
position: relative;
top: 20px;
left: 10px;
}

3. POSITION: ABSOLUTE :

>>>> Definition:
         >> Absolute positioning positions an element relative to the nearest positioned ancestor.

>>>> Characteristics:

**** Removed from normal document flow

**** No space is reserved

**** Positioned using top, left, right, bottom

**** If no positioned parent exists, it uses the body

**** Example:
.parent {
position: relative;
}

.child {
position: absolute;
top: 0;
right: 0;
}

>>>> Use cases:

**** Dropdown menus

T*** ooltips

**** Badges

**** Icons on images

**** Key Rule:

>>>> Absolute always looks for the nearest parent with:
position: relative | absolute | fixed | stickyExample:


>>>> POSITION: FIXED :

>>>> Definition:

        >. Fixed positioning positions an element relative to the viewport (screen).

>>>> Characteristics:

**** Removed from document flow

**** Stays fixed even when scrolling

**** Positioned using top, left, right, bottom

**** Example:
.navbar {
position: fixed;
top: 0;
width: 100%;
}


>>>> Note:
     >.. Fixed elements do not move when the page scrolls.


>>>> POSITION: STICKY :

>>>> Definition:
        >>. Sticky positioning is a mix of relative and fixed.

>>>> Characteristics:

**** Behaves like relative initially

**** Becomes fixed when scrolling reaches a point

**** Requires top, left, etc. to work

**** Example:
.header {
position: sticky;
top: 0;
}

**** Use cases:

Sticky headers

Table headings

Section titles

***** Important:

Sticky works only if:
      Parent does not have overflow: hidden

       A top value is defined

>>>>>>>>>> INTRODUCTION

      >>. CSS Transform is used to change the shape, position, size, or orientation of an element.

      >>. Transform does NOT affect surrounding elements. Only the element itself changes visually.

--->> Syntax:

1. ROTATE ELEMENTS :

     >.. Rotate is used to rotate an element around its center.

-->> Syntax:

>. Example:

-->> Positive value â†’ clockwise
-->> Negative value â†’ anticlockwise

2. TRANSLATE ELEMENTS :

           >>. Translate is used to move an element from its original position.

-->> Syntax:

-->> translateX(px) â†’ moves left or right
-->> translateY(px) â†’ moves up or down

>.. Example:
transform: translateX(50px);

3. SCALE ELEMENTS :

     >>. Scale is used to increase or decrease the size of an element.

-->> Syntax:

-->> scale(1) â†’ normal size
-->> scale(1.2) â†’ bigger
-->> scale(0.8) â†’ smaller

>>.. Example:
transform: scale(1.1);

4. SKEW ELEMENTS :

       >.. Skew is used to tilt an element along X or Y axis.

-->> Syntax:

-->> skewX() â†’ tilts horizontally
-->> skewY() â†’ tilts vertically

-->> Example:

5. USE OF MATRIX FUNCTION :

         >.. Matrix is an advanced way to combine multiple transforms in one function.

>>.. IT controls:

-->> scale

-->> rotate

-->> translate

-->> skew

>>>>> Syntax:

-----------------------------------------------------------------
Note:
